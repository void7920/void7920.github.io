# Design Source

## Structual modeling

```verilog
module XNOR_Structural(
	y,
	a,
	b
	);
     
	output y;
	input a;
	input b;

//            o, i, i, ...
	xnor( y, a, b );
endmodule
```

## Dataflow modeling

```verilog
module XNOR_Dataflow(
	y,
	a,
	b
	);
     
	output y;
	input a;
	input b;

	assign y = ~( a ^ b );
endmodule
```

## Behavioral modeling

```verilog
module XNOR_behavioral(
	y,
	a,
	b
	);
     
	output reg y;
	input a;
	input b;

	case( { b, a } )
		2'b00: y = 1'b1;
		2'b01: y = 1'b0;
		2'b10: y = 1'b0;
		2'b11: y = 1'b1;
	endcase
endmodule
```

	end
endmodule
